# 🌍 CarbonChain - Supply Chain Carbon Tracker

A mobile app that helps companies track and reduce carbon emissions across their entire supply chain.

## 🎯 What It Does

**CarbonChain** enables companies to:
- 📊 Track supplier carbon footprints in real-time
- 🎯 Identify top polluters automatically
- 📈 Set and monitor reduction targets
- 💡 Get actionable optimization recommendations

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/kunalprooo/carbonchain.git
cd carbonchain

# Install dependencies
npm install

# Start the app
npm start

# Press 'w' for web, 'i' for iOS, 'a' for Android
```

## 📱 Features

✅ **Real-time Carbon Calculations** - Instant emissions tracking  
✅ **Supplier Ranking** - See which suppliers pollute the most  
✅ **20% Reduction Targets** - Automatic goal setting  
✅ **Beautiful UI** - Green theme, professional design  
✅ **Cross-platform** - Works on iOS, Android, and Web  

## 💡 Use Case

**Before CarbonChain:**
- Nike needs $100K consultants to audit suppliers
- Takes months to identify carbon hotspots
- No real-time visibility into emissions

**With CarbonChain:**
- Nike adds suppliers in seconds
- Instantly sees Foxconn is 40% of emissions
- Reduces carbon in weeks, not months

## 🏗️ Tech Stack

- **Frontend:** Expo + React Native (cross-platform mobile)
- **Language:** TypeScript
- **Database:** Ready for Neo4j integration (phase 2)
- **Styling:** React Native StyleSheet
